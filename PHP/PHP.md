### 什么是 PHP？
* PHP 是 "PHP Hypertext Preprocessor" 的首字母缩略词
* PHP 是一种被广泛使用的开源脚本语言
* PHP 脚本在 __服务器__ 上执行
* PHP 没有成本，可供免费下载和使用


### 什么是 PHP 文件？
* PHP 文件能够包含文本、HTML、CSS 以及 PHP 代码
* PHP 代码在服务器上执行，而结果以纯文本返回浏览器
* PHP 文件的后缀是 ".php"
### PHP 整数
整数是没有小数的数字。

整数规则：

* 整数必须有至少一个数字（0-9）
* 整数不能包含逗号或空格
* 整数不能有小数点
* 整数正负均可
* 可以用三种格式规定整数：十进制、十六进制（前缀是 0x）或八进制（前缀是 0）
